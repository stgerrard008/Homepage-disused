Compile the HiWiFi System

Recently I got a HiWiFi on which I attempted to accomplish some function. HiWiFi, a wireless network manager that provides automatic connection to WLAN and open hotspots, can be considered as a smart router in brief. The system of HiWiFi is based on an open source project OpenWrt which I arrange to expound in later articles. In the rest of this article, I will detail the steps how to compile the source code of HiWiFi embedded system instead of OpenWrt.

Tip: HiWiFi 1 remains the only version within HiWiFi series that support OpenWrt, even though the HiWiFi system is developed on OpenWrt.
